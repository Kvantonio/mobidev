# Mobidev minitasks

## Task1
There are two files in the folder `task1`:
* script.sh
* task1.rb

To run the ```./task1/script.sh```:

```sh
zsh script.sh sometext
```
To run the ```./task1/task1.rb```:

```sh
ruby ./task1.rb
```
